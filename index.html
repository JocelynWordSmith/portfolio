Hunger Crunch

  • Contact Josh
  • View Code

As my final project for the development class I took at The Iron Yard Academy in Greenville, SC, I developed a web app for a game called hunger crunch. The game can currently be found at www.HungerCrunch.com, or on the IOS App Store. The app is scheduled to get a test run with about 700 users in September of 2014, after which it will go live for regular use.

The team I worked with was given designs to help us stay in line with the existing site's styles, and we used parse.com to manage our data, so all the information displayed is live and dynamic.

The purpose of the app is to give youth group leaders and teachers a way of managing tournament scores based on the incoming player data in the game, as well as a way to manage what players were in which groups in order to have competitions betweens grades or small groups.

The languages used were HTML, CSS (scss), and JavaScript, and were supported by jQuery, Underscore, and Parse (a backbone derivative). List.js was used to manage list filtering, and pickadate.js was used for interactive calendars to choose start and end dates for the tournaments. Modernizer and sass were also used.

My team was successful in building this app in two and a half weeks despite using new plugins and cloud code functions to manage the score totaling on multiple levels (total score, group scores, and player scores). There is also login functionality, and ACL's attached to certain objects to ensure that only the group leaders can change their team members, and also dynamic options that will only show up if the person with access is the one viewing the page.
